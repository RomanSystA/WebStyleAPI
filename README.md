Данная интеграция призвана упростить процесс взаимодействия компании с подрядчиками путем внедрения в CRM сервиса MadeTask. С помощью него компании не придется взаимодействовать с подрядчиками через мессенджеры и производить оплату работ по СБП: всё это можно будет делать прямо в корпоративной CRM.
